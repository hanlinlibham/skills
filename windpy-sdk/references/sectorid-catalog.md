申万行业需要通过**两步查询**获取成分股：

```python
result = w.wset("sectorconstituent", "date=20260209;windcode=801010.SI")

```

## 使用示例

```python
from WindPy import w
w.start()

# 获取全部A股成分股列表
data = w.wset("sectorconstituent", "date=20260108;sectorid=a001010100000000")
# 返回: date, wind_code, sec_name
stocks = data.Data[1]  # wind_code 列表

# 获取沪深300成分股
err, df = w.wset("sectorconstituent", 
    "date=20260108;sectorid=100000009800000000", usedf=True)

# 获取板块历史序列数据
err, df = w.wses("1000000098000000", "sec_close_avg",
    "2024-01-01", "2024-06-30", "", usedf=True)

# 多板块截面数据对比
err, df = w.wsee("a001010200000000,a001010300000000",
    "sec_close_avg,sec_pct_chg_avg,sec_pe_ttm_median",
    "tradeDate=20260108", usedf=True)

# 获取申万行业成分股
err, df = w.wset("sectorconstituent",
    "date=20260108;sectorid=a39901130g000000", usedf=True)  # 食品饮料

w.stop()
```

## 获取板块成分股的函数

```python
def get_sector_constituents(sectorid, date=None):
    """
    获取板块成分股列表
    
    Parameters:
    -----------
    sectorid : str
        板块代码，如 '1000000098000000' (沪深300)
    date : str, optional
        日期，格式 '20260108'，默认当前日期
    
    Returns:
    --------
    list : wind_code 列表
    """
    if date is None:
        from datetime import datetime
        date = datetime.now().strftime('%Y%m%d')
    
    data = w.wset("sectorconstituent", f"date={date};sectorid={sectorid}")
    if data.ErrorCode == 0 and len(data.Data) > 1:
        return data.Data[1]  # wind_code 列表
    return []

# 使用示例
hs300 = get_sector_constituents('1000000098000000')
food_beverage = get_sector_constituents('a39901130g000000')  # 食品饮料
```

## 注意事项

1. **日期格式：** wset 使用 `date=YYYYMMDD`，wss 使用 `tradeDate=YYYYMMDD`
2. **成分股变化：** 指数成分股定期调整，历史数据需指定日期
3. **申万行业：** 每年会有行业分类调整，2021年后使用申万2021版分类
4. **数据权限：** 部分板块（如科创板、北交所）需要相应权限
