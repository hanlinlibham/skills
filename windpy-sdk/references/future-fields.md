# WindPy 期货专用字段参考

期货数据通过 `w.wsd()`、`w.wss()`、`w.wsi()`（分钟）、`w.wsq()`（实时）获取。

## 期货基础行情字段

| 字段 | 含义 | 说明 |
|------|------|------|
| `close` | 收盘价 | 日收盘价 |
| `settle` | 结算价 | 当日结算价 |
| `pre_settle` | 昨结算价 | 上一交易日结算价 |
| `open` | 开盘价 |  |
| `high` | 最高价 |  |
| `low` | 最低价 |  |
| `volume` | 成交量 |  |
| `amt` | 成交额 |  |

## 期货特有字段

| 字段 | 含义 | 说明 |
|------|------|------|
| `oi` | 持仓量 | Open Interest |
| `oi_chg` | 持仓量变化 | 较上一交易日变化 |
| `limit_up` | 涨停价 |  |
| `limit_down` | 跌停价 |  |
| `margin` | 保证金比例 | 交易所保证金要求 |

---

## 使用示例

```python
from WindPy import w
w.start()

# 期货日级数据（主力合约）
err, df = w.wsd("IF00.CFE", "open,high,low,close,settle,oi,oi_chg", 
                "-10D", "", "", usedf=True)

# 股指期货截面数据
err, df = w.wss("IF00.CFE,IH00.CFE,IC00.CFE,IM00.CFE",
    "sec_name,close,settle,oi,limit_up,limit_down",
    "", usedf=True)

# 期货分钟数据
err, df = w.wsi("IF00.CFE", "open,high,low,close,volume,oi",
    "2026-02-06 09:30:00", "2026-02-06 15:00:00", "BarSize=5", usedf=True)

# 期货实时行情
err, df = w.wsq("IF00.CFE", "rt_last,rt_settle,rt_oi,rt_high,rt_low", usedf=True)

w.stop()
```

## 期货代码格式

| 市场 | 后缀 | 示例 | 说明 |
|------|------|------|------|
| 中金所 | `.CFE` | `IF00.CFE` | 股指期货主力合约 |
| 上期所 | `.SHF` | `RB00.SHF` | 螺纹钢主力 |
| 大商所 | `.DCE` | `I00.DCE` | 铁矿石主力 |
| 郑商所 | `.CZC` | `MA00.CZC` | 甲醇主力 |
| 上期能源 | `.INE` | `SC00.INE` | 原油主力 |
| 广期所 | `.GFE` | `LC00.GFE` | 碳酸锂主力 |

### 主力合约 vs 具体合约

- **主力合约：** `IF00.CFE`、`RB00.SHF` - 自动映射当前最活跃的合约
- **具体合约：** `IF2503.CFE`、`RB2505.SHF` - 指定交割月份

## 注意事项

1. **主力合约切换：** 每月第三个周五后自动切换到下月合约
2. **交易时间：** 期货有夜盘，日盘 9:00-15:00，部分品种有 21:00-次日 2:30 夜盘
3. **结算价：** 期货使用结算价计算盈亏，而非收盘价
4. **持仓量：** `oi` 反映市场参与度和资金流向
5. **代码后缀：** 注意区分 `.CFE`（中金所）、`.SHF`（上期所）等
